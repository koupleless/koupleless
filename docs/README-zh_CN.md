## Install dependencies:

- [node](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/get-npm)
- [hugo](https://github.com/gohugoio/hugo/releases).

## install postCSS
``
npm install -D autoprefixer
npm install -D postcss-cli

npm install -D postcss
``

## start
```shell
npm install
hugo server
```
## Test 
```shell
curl localhost:1313
```
